# FiveMクラフトレシピ研究 — 研究報告書

東京国際大学 商学部 データサイエンスコース 竹田ゼミ（データアナリティクスゼミ）の学生が、専門演習 データサイエンス研究の研究報告書および発表資料を作成するための Quarto テンプレートです。

FiveMゲーム内のクラフトレシピを対象に、材料・労働ポイント・カテゴリごとの分析をまとめた研究報告書です。

## ディレクトリ構成

```
.
├── index.qmd              # 目次ページ
├── 01_手芸クラフト.qmd     # 第1章：手工芸カテゴリのクラフトレシピ一覧
├── 02_武器クラフト.qmd     # 第2章：武器カテゴリのクラフトレシピ一覧
├── 03_受注場所.qmd         # 第3章：受注場所の分布・マップ
├── 04_組織関連.qmd         # 付章：組織関連レシピ
├── a1_発表資料.qmd         # 発表スライド（reveal.js）
├── _quarto.yml             # Quarto 設定ファイル
├── data/                   # 分析データ
├── src/                    # データ生成・前処理スクリプト
├── fig/                    # 外部作成の図表・画像
├── ref/                    # 参考文献（BibTeX）
├── css/                    # カスタムスタイルシート
└── docs/                   # 出力ディレクトリ（HTML）
```

## 前提環境

- [Quarto](https://quarto.org/) (>= 1.4)
- [R](https://www.r-project.org/)

## ビルド方法

```bash
# 研究報告書のビルド
quarto render

# 研究報告書のプレビュー
quarto preview

# 発表資料のみビルド
quarto render a1_発表資料.qmd
```

> **Note:** Windowsでは `make` コマンドが使えないため、`quarto render` を直接使用してください。

`quarto render` を実行すると研究報告書（Book形式 HTML）と発表資料（reveal.js HTML）が `docs/` ディレクトリに出力されます。

## 使い方

1. このリポジトリをクローンします
2. `_quarto.yml` の `title` と `author` を自分の研究テーマ・名前に変更します
3. 各 `.qmd` ファイルを編集して研究内容を記述します
4. 分析データは `data/` に配置し、前処理スクリプトは `src/` に配置します
5. `quarto render` でビルドし、`docs/` の出力を確認します
6. GitHub Pagesで公開する場合は、リポジトリの Settings → Pages → Branch: main / docs フォルダを選択
